ElDewrito Dedicated Server Status

Currently, this website is home to a simple tool to generate a status banner image for your Eldewrito server.

Please note, this tool is currently in BETA status, and further improvements are planned (including an actual website.)

How To:

You may construct the banner URL for your server by knowing the IP address and port of the server.

Entering the port is optional, if your game is running on the default port 11775

https://eldewritoserver.com/banner/IP:PORT

Example (with default port): https://eldewritoserver.com/banner/1.2.3.4

Example (with custom port): https://eldewritoserver.com/banner/1.2.3.4:12345
